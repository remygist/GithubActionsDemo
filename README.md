# Github Actions Demo
This repository demonstrates how to use GitHub Actions to automate workflows for deployment, testing, and custom actions. The project includes simple examples of automating tasks such as deploying a React app, running tests, and fetching random jokes using external APIs.

# Requirements
- [Node](https://nodejs.org/en)
- [Vite](https://vite.dev/)
- [React](https://react.dev/)
- Github account

# Features
- Automated Deployment: Deploys a React application automatically when changes are pushed to the repository.
- Test Automation: Runs tests on every push or pull request to the main branch.
- Custom Actions: Demonstrates how to use custom GitHub Actions, like fetching random jokes via an API.

# Setup
1. **Clone the repository**
2. **Install dependencies**
   ```sh
   npm install
   ```
3. Push code to the repository to see the actions take place.
